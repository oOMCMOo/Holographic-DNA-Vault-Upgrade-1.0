# Holographic DNA Vault by oOMCMOo

A state-of-the-art biological data storage and encryption interface. This application leverages three-dimensional holographic visualization, advanced cryptography, and Web3 integration to provide a secure environment for sensitive digital assets.

## 🚀 Key Features

### 1. 3D Holographic Vault (Core)
- **Interactive Helix:** A high-quality 3D DNA double helix rendered using Three.js and React Three Fiber.
- **Enhanced Visuals:** Data flux particles, additive blending effects, and atmospheric lighting.
- **Micro-Interaction:** Hover over individual nucleotides (A, T, G, C) to view molecular data.
- **Sequence Search:** Integrated search bar that identifies and highlights specific genetic sequences (e.g., "ATGC") directly within the 3D structure.
- **Dynamic Camera:** Orbit, zoom, and pan around the vault core for detailed inspection.

### 2. Molecular Mapping (New)
- **Genomic Grid:** Visualize DNA strands as a high-density color-coded grid.
- **Interactive Base Mapping:** Individual base-pair inspection with micro-tooltips.
- **Density Indexing:** Structural analysis reports for stored digital genomes.

### 3. Post-Quantum Cipher Engine
- **Military-Grade Encryption:** Utilizes AES-256 for simulated quantum-resistant security.
- **Passphrase Protection:** Encrypt raw text or binary data into DNA-encoded ciphers.
- **Secure Decryption:** Fragment decryption requires local identity keys (passphrases).

### 3. Molecular File Synthesizer
- **Format Support:** Convert files (MP3, ZIP, MP4, MOV, etc.) into biological sequences.
- **Binary-to-DNA Mapping:** Advanced algorithm that translates 0/1 bitsets into A/T/G/C base pairs.
- **Vault Ingestion:** Synthesized strands are staged for long-term molecular storage.

### 4. Web3 Secure Bridge
- **Wallet Connectivity:** Securely attach Ethereum-compatible wallets (MetaMask, etc.) via ethers.js.
- **Asset Monitoring:** Real-time balance tracking and secure vault registry for decentralized assets.
- **Multi-Functional Storage:** Bridges physical (biological DNA representation) and digital (blockchain) asset management.

### 5. Cybernetic Interface (HUD)
- **Sacred Geometry Design:** Aesthetics inspired by the Flower of Life and Metatron's Cube.
- **Real-time Monitoring:** HUD panels displaying capacity, integrity, system entropy, and ambient nebula flux.
- **Biotech Aesthetic:** High-contrast violet and emerald themes with glitch-text effects and scanline overlays.

## 🛠 Specifications

- **Frontend:** React 18, Vite, TypeScript.
- **Styling:** Tailwind CSS (Modern v4 implementation).
- **3D Engine:** Three.js, React Three Fiber, R3F Drei.
- **Animations:** Motion (Framer Motion).
- **Cryptography:** Crypto-js.
- **Web3:** Ethers.js.
- **Icons:** Lucide-react.
